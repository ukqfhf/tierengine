# Tier Engine 🏆

A simple, static-first tier list engine built with **11ty** and **Markdown**.

Tier Engine helps you create classic vertical tier lists that are easy to **edit**, **fork**, and **deploy**.  
It is designed to be lightweight, portable, and friendly for both creators and developers. ✨

## What is this? 🤔

This project is a small engine for building tier lists as static pages.

That means your tier lists are easy to:

* keep in a repository
* version with Git
* deploy to GitHub Pages
* customize to fit your own style

## Why is it useful? 💡

Tier Engine is useful because it gives you a **simple and portable format** for tier lists.

With it, you can:

* create shareable tier lists without a database
* keep all content in your own repo
* edit tiers and items in a human-readable format
* host your list for free as a static site
* build custom themed tier lists for games, characters, tools, rankings, and more

## Current features 🚀

* classic vertical tier board
* Markdown-based tier definition
* item pool for unranked items
* drag and drop in the browser
* static build with 11ty
* GitHub Pages deployment support
